## Overview

__Trinomial Calculator__ is a .NET-based desktop application designed to help users calculate and solve trinomial equations of the form ax^2 + bx + c = 0. 
The application allows users to enter coefficients, and it will compute the roots of the equation using the quadratic formula. 
It’s an easy-to-use tool for solving quadratic equations.
